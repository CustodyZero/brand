Assets are provided in SVG (preferred) and PNG at 1x, 2x, and 3x for standard use cases. Do not use rasterized assets at sizes that cause visible degradation — use SVG.

## Quick Rules

**Do** use these assets to accurately identify CustodyZero and its products in editorial, press, and integration contexts.

**Do not** modify, recolor, distort, or combine these marks with other elements.

**Do not** use these assets in your own product branding, merchandise, or promotional materials without written permission.

**Do not** assume the Apache 2.0 license on Archon or any other CustodyZero open source project extends to brand assets. It does not.

The full permitted and prohibited use policy is in [LICENSE](LICENSE).
